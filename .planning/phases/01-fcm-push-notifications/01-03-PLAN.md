1. **DTOs** — All in `kz.innlab.template.notification.dto`, all as `data class` with jakarta.validation:

- `RegisterTokenRequest`: `platform: String` (validated: must be ANDROID, IOS, or WEB), `fcmToken: @NotBlank String`, `deviceId: @NotBlank String`
- `DeviceTokenResponse`: `id: UUID`, `platform: String`, `fcmToken: String`, `deviceId: String`, `createdAt: Instant`, `updatedAt: Instant`. Add `companion object { fun from(token: DeviceToken): DeviceTokenResponse }` factory.
- `SendToTokenRequest`: `token: @NotBlank String`, `title: @NotBlank @Size(max=255) String`, `body: @NotBlank String`, `data: Map<String, String> = emptyMap()`
- `SendMulticastRequest`: `tokens: @NotEmpty @Size(max=500) List<@NotBlank String>`, `title: @NotBlank @Size(max=255) String`, `body: @NotBlank String`, `data: Map<String, String> = emptyMap()`
- `SendToTopicRequest`: `topic: @NotBlank String`, `title: @NotBlank @Size(max=255) String`, `body: @NotBlank String`, `data: Map<String, String> = emptyMap()`
- `NotificationHistoryResponse`: `id: UUID`, `type: String`, `recipient: String`, `title: String`, `body: String`, `status: String`, `createdAt: Instant`. Add `companion object { fun from(history: NotificationHistory): NotificationHistoryResponse }` factory.
- `TopicSubscribeRequest`: `token: @NotBlank String`
- `TopicResponse`: `id: UUID`, `name: String`, `createdAt: Instant`. Add `companion object { fun from(topic: NotificationTopic): TopicResponse }` factory.

2. **NotificationController.kt** — `@RestController` `@RequestMapping("/api/v1/notifications")` in `kz.innlab.template.notification.controller`:
- Constructor injection: `DeviceTokenService`, `NotificationService`, `TopicService`
- Extract userId from JWT: `@AuthenticationPrincipal jwt: Jwt` -> `UUID.fromString(jwt.subject)`

**Token management endpoints:**
- `@PostMapping("/tokens") fun registerToken(@Valid @RequestBody request: RegisterTokenRequest, @AuthenticationPrincipal jwt: Jwt): ResponseEntity<DeviceTokenResponse>`:
  - Parse platform: `Platform.valueOf(request.platform.uppercase())`
  - Call `deviceTokenService.register(userId, platform, request.fcmToken, request.deviceId)`
  - Return `ResponseEntity.status(201).body(DeviceTokenResponse.from(token))`

- `@GetMapping("/tokens") fun listTokens(@AuthenticationPrincipal jwt: Jwt): ResponseEntity<List<DeviceTokenResponse>>`:
  - Return 200 with `deviceTokenService.listByUser(userId).map { DeviceTokenResponse.from(it) }`

- `@DeleteMapping("/tokens/{deviceId}") fun deleteToken(@PathVariable deviceId: String, @AuthenticationPrincipal jwt: Jwt): ResponseEntity<Void>`:
  - Call `deviceTokenService.deleteByDeviceId(userId, deviceId)`
  - Return `ResponseEntity.noContent().build()`

- `@DeleteMapping("/tokens") fun deleteAllTokens(@AuthenticationPrincipal jwt: Jwt): ResponseEntity<Void>`:
  - Call `deviceTokenService.deleteAllByUser(userId)`
  - Return `ResponseEntity.noContent().build()`

**Send endpoints (all return 202 Accepted):**
- `@PostMapping("/send/token") fun sendToToken(@Valid @RequestBody request: SendToTokenRequest, @AuthenticationPrincipal jwt: Jwt): ResponseEntity<Map<String, UUID>>`:
  - `val historyId = notificationService.sendToToken(userId, request.token, request.title, request.body, request.data)`
  - Return `ResponseEntity.accepted().body(mapOf("notificationId" to historyId))`

- `@PostMapping("/send/multicast") fun sendMulticast(@Valid @RequestBody request: SendMulticastRequest, @AuthenticationPrincipal jwt: Jwt): ResponseEntity<Map<String, UUID>>`:
  - `val historyId = notificationService.sendMulticast(userId, request.tokens, request.title, request.body, request.data)`
  - Return `ResponseEntity.accepted().body(mapOf("notificationId" to historyId))`

- `@PostMapping("/send/topic") fun sendToTopic(@Valid @RequestBody request: SendToTopicRequest, @AuthenticationPrincipal jwt: Jwt): ResponseEntity<Map<String, UUID>>`:
  - Validate topic exists: `topicService.validateTopicExists(request.topic)`
  - `val historyId = notificationService.sendToTopic(userId, request.topic, request.title, request.body, request.data)`
  - Return `ResponseEntity.accepted().body(mapOf("notificationId" to historyId))`

**Topic subscription endpoints:**
- `@PostMapping("/topics/{name}/subscribe") fun subscribe(@PathVariable name: String, @Valid @RequestBody request: TopicSubscribeRequest, @AuthenticationPrincipal jwt: Jwt): ResponseEntity<Void>`:
  - `topicService.subscribe(request.token, name)`
  - Return `ResponseEntity.ok().build()`

- `@DeleteMapping("/topics/{name}/subscribe") fun unsubscribe(@PathVariable name: String, @Valid @RequestBody request: TopicSubscribeRequest, @AuthenticationPrincipal jwt: Jwt): ResponseEntity<Void>`:
  - `topicService.unsubscribe(request.token, name)`
  - Return `ResponseEntity.ok().build()`

**History endpoint:**
- `@GetMapping("/history") fun getHistory(@RequestParam(required = false) cursor: UUID?, @RequestParam(defaultValue = "20") size: Int, @AuthenticationPrincipal jwt: Jwt): ResponseEntity<List<NotificationHistoryResponse>>`:
  - `val history = notificationService.getHistory(userId, cursor, size.coerceIn(1, 100))`
  - Return 200 with `history.map { NotificationHistoryResponse.from(it) }`
