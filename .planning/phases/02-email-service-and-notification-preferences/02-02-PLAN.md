    2. **MailDispatcher.kt** — Create in `notification/service/`:
       - Follow exact NotificationDispatcher pattern: separate `@Service` bean with `@Async @Transactional` methods
       - Constructor injects: `JavaMailSender`, `MailHistoryRepository`, `MailProperties`
       - `@Async @Transactional dispatchEmail(historyId: UUID, to: String, subject: String, textBody: String?, htmlBody: String?, attachments: List<EmailAttachment>)`:
         1. Load MailHistory by historyId
         2. Retry loop: `for (attempt in 1..mailProperties.retry.maxAttempts)`
            - Create MimeMessage via `javaMailSender.createMimeMessage()`
            - Create `MimeMessageHelper(message, attachments.isNotEmpty())` (multipart only if attachments)
            - Set from: `mailProperties.smtp.from`
            - Set to, subject, text (htmlBody if present with isHtml=true, else textBody)
            - For each attachment: `helper.addAttachment(att.filename, ByteArrayResource(att.bytes), att.contentType)`
            - Call `javaMailSender.send(message)`
            - Update history: status = SENT, attempts = attempt
            - Save and return (success)
            - On exception: log warning, increment attempts on history, if not last attempt: `Thread.sleep(mailProperties.retry.delayMs)`
         3. After all retries exhausted: history.status = FAILED, save
       - `fun sendDirect(to: String, subject: String, textBody: String)` — non-async, non-tracked send for verification codes. Uses JavaMailSender directly with SimpleMailMessage or MimeMessage. No retry (codes are time-sensitive).

       - All methods open IMAP Store per-request in try/finally (NEVER hold as singleton — documented decision)
       - Private helper: `fun <T> withInbox(readWrite: Boolean = false, block: (Folder) -> T): T` that:
         1. Checks `mailProperties.imap.host.isNotBlank()` — throws `IllegalStateException("IMAP not configured")` if blank
         2. Creates `Properties` with `mail.imaps.host`, `mail.imaps.port`, SSL settings
         3. Gets Session, Store (`imaps` or `imap` based on sslEnabled)
         4. Connects with username/password
         5. Opens INBOX folder (readWrite if needed)
         6. Calls block(folder)
         7. Closes folder and store in finally block
       - `fun listInbox(offset: Int, size: Int, unreadOnly: Boolean): InboxPage`:
         - Uses `withInbox(readWrite = false)` block
         - If unreadOnly: use `folder.search(FlagTerm(Flags(Flags.Flag.SEEN), false))` to get unread messages
         - Else: get all messages from folder
         - Total count from messages array size
         - Apply offset/size pagination: slice the array (newest first)
         - For each message, extract: messageNumber, subject, from (first InternetAddress), date (sentDate), isRead (SEEN flag), hasAttachments (check multipart disposition)
         - Return `InboxPage(messages: List<InboxMessage>, total: Int, offset: Int, size: Int)`
         - Use FetchProfile with ENVELOPE and FLAGS for performance
       - `fun getMessage(messageNumber: Int): EmailMessage`:

         - Gets message by number: `folder.getMessage(messageNumber)`
         - Extracts: subject, from, to, date, textBody (find text/plain part), htmlBody (find text/html part), attachments metadata list (filename, contentType, size for each Part with ATTACHMENT disposition)
         - Returns `EmailMessage(messageNumber, subject, from, to, date, textBody, htmlBody, attachments, isRead)`

       - Helper for extracting body parts from multipart messages:
         ```kotlin
         private fun extractParts(part: Part, textParts: MutableList<String>, htmlParts: MutableList<String>, attachments: MutableList<AttachmentMeta>) {
             if (part.isMimeType("multipart/*")) {
                 val mp = part.content as Multipart
                 for (i in 0 until mp.count) extractParts(mp.getBodyPart(i), textParts, htmlParts, attachments)
             } else if (part.isMimeType("text/plain") && part.disposition == null) {
                 textParts.add(part.content as String)
             } else if (part.isMimeType("text/html") && part.disposition == null) {
                 htmlParts.add(part.content as String)
             } else if (part.disposition != null) {
                 attachments.add(AttachmentMeta(part.fileName ?: "unknown", part.contentType.split(";")[0], part.size))
             }
         }
         ```
