# Technology Stack

**Analysis Date:** 2026-03-01

## Languages

**Primary:**
- Java 24 - JVM runtime environment
- Kotlin 2.2.21 - Primary application language

**Build & Compilation:**
- Kotlin compiles to bytecode for JVM execution

## Runtime

**Environment:**
- Java 24 (OpenJDK or JDK 24+)
- Spring Boot 4.0.3 (parent POM version)

**Package Manager:**
- Maven 3.x (Apache Maven)
- Lockfile: `mvnw` wrapper included for reproducible builds

## Frameworks

**Core:**
- Spring Boot 4.0.3 - Full-stack web framework
  - spring-boot-starter-webmvc - REST and MVC support
  - spring-boot-starter-parent - Dependency management and defaults

**Serialization/JSON:**
- Jackson (spring-boot-starter-webmvc includes jackson-databind)
  - jackson-module-kotlin - Kotlin-specific Jackson support for proper handling of Kotlin types

**Testing:**
- JUnit 5 (via spring-boot-starter-test)
  - kotlin-test-junit5 - Kotlin testing utilities for JUnit 5
  - spring-boot-test - Spring test context support

**Build/Dev:**
- spring-boot-maven-plugin - Spring Boot Maven integration for packaging and running
- kotlin-maven-plugin - Kotlin compilation support
  - kotlin-maven-allopen - Compiler plugin for Spring AOP compatibility

## Key Dependencies

**Critical:**
- spring-boot-starter-webmvc - Provides servlet-based web stack, includes Tomcat, Spring Web, Spring MVC
- kotlin-stdlib - Kotlin standard library
- kotlin-reflect - Runtime reflection support for Kotlin
- jackson-module-kotlin - Ensures proper JSON serialization/deserialization of Kotlin data classes

**Data Access:**
- postgresql (runtime scope) - PostgreSQL JDBC driver for database connectivity

## Configuration

**Environment:**
- Main configuration: `src/main/resources/application.yaml`
- Environment variables supported via Spring Boot's externalized configuration
- Env template provided: `.env.template` (empty - for future expansion)

**Build:**
- `pom.xml` - Maven project object model
- Kotlin compiler arguments configured for strict JSR-305 nullability and Spring annotation defaults

## Platform Requirements

- Java 24 JDK or later

**Production:**
